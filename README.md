#Mozaïk demo dashboards

This repository shows how to use Mozaïk with npm extensions.

## Installation

### Clone the repo

```bash
git clone git@github.com:plouc/mozaik-demo.git
```

### Install packages

```bash
cd mozaik-demo
npm install
```

### Publish assets (js/css)

```bash
gulp publish
```

### Run the app

```bash
node app.js
```
